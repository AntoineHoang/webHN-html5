Design

Combinaisons de couleurs:
Un arrière plan avec une texture de textile gris foncé, souvent accompagné d'un "main" noir mat pour pour un effet chic. Nous avons ensuite pris les couleurs dorée et argentée pour les onglets et les titres, ce qui donne un effet de prestige. Doré est bien à l'opposé des deux couleurs foncées d'arrière plan dans le cercle chromatique. Ce qui nous donne une tétrade, deux paires de couleurs complémentaires.

Modèle de page type:
Le modèle de page type dont nous nous sommes servis est sauvegarder sous le nom prototype.html dans le fichier. Voici un lien pour y accéder: Prototype

Police:
Nous avons choisi la police Cursive pour nos titres, une police qui est semi en lettres attachées pour accompagnée le thème du site: propre, classe et dispendieux. Le contenu de l'information est en Helvetica, Arial, Sans-Serif vu que c'est informatif. Dans l'onglet menu, nous avons mit les images et le texte en diagonal pour attirer le client avec les critères V++ vus en classe.
